moment.js보다 용량이 훨씬 가벼우며, 사용하는 방법도 유사하므로 npm trend에서 확인해보면 많은 개발자가 선택하는 라이브러리임을 확인가능함.

필요한 것들을 extend(확장)로 불러와서 사용하면 되는 방식이다.
